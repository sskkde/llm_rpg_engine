# 青岚仙侠世界 - 内容包

## 概述

青岚仙侠世界是一个以修仙试炼为主题的内容包，包含完整的宗门设定、NPC角色、任务系统和剧情线索。

## 内容清单

### 世界设定
- **青岚仙门**：千年仙门，以剑道与炼丹闻名，外门弟子需通过试炼方可入内门

### 地点 (11个)
- 青岚山门 - 宗门入口
- 弟子广场 - 外门弟子聚集之所
- 试炼场 - 试炼关卡所在地
- 试炼堂 - 领取试炼任务
- 外门居所 - 弟子起居之所
- 药园 - 灵草种植区
- 藏经阁外区 - 基础功法查阅
- 灵林试炼区 - 主要试炼区域
- 崖边祭坛 - 古老祭坛
- 秘境入口 - 隐秘石门
- 异变核心 - 试炼异变源头

### NPC角色 (3个)
- **清玄上人** - 宗门护法长老，暗中守护试炼真相
- **李长老** - 试炼长老，试炼异变的幕后策划者
- **柳师姐** - 外门师姐，暗中传递信息

### 势力 (2个)
- **青岚宗** - 正统修仙势力
- **暗影势力** - 隐秘派系，策划试炼异变

### 任务 (2个)
- **初入青岚** - 主线任务，通过基础试炼
- **异变之谜** - 支线任务，发现试炼异变真相

### 物品 (4个)
- **聚气丹** - 入门丹药，提升修炼效率
- **御风符** - 提升身法速度的符箓
- **灵石** - 修仙界通用货币
- **青岚令牌** - 开启秘境的关键道具

### 剧情节点 (3个)
- **试炼异变初现** - 发现异变线索
- **长老的异样** - 对李长老产生怀疑
- **秘境入口现世** - 解锁秘境入口

### 提示模板 (2个)
- 叙事提示模板 - 仙侠风格叙事
- NPC对话提示模板 - 角色对话生成

## 使用说明

1. 将此内容包放入 `content_packs/` 目录
2. 通过管理后台导入内容包
3. 创建新游戏时选择"青岚仙侠世界"
